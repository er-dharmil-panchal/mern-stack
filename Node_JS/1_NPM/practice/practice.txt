3. Script Master

     solution - (add this to the 'scripts' in package.json)
          - "scripts": {
                         "dev": "node app.js" (any basic js file)
                    }
          - (now run npm run dev)

4. Version Check - Check installed packages via terminal:

     npm list
     # Or for just the top level:
     npm list --depth=0

5. The Clean Up - To remove a package completely:

     npm uninstall lodash
